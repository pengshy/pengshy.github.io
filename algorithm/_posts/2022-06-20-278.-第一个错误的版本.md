### 278. 第一个错误的版本

#### 二分模板题

```c++
// The API isBadVersion is defined for you.
// bool isBadVersion(int version);

class Solution {
public:
    int firstBadVersion(int n) {
        int l = 1, r = n;
        while (l < r)
        {
            int mid = l + (r - l) / 2;
            if (!isBadVersion(mid)) l = mid + 1;
            else r = mid;
        }
        return l;
    }
};
```

#### 时间复杂度：$$ O(logn) $$

#### 空间复杂度：$$ O(1) $$
